# Yearn V3 — DeFiScan Protocol Review
 
This repository contains an independent decentralization review of the Yearn V3 (yVaults) protocol on Ethereum Mainnet.
 
The analysis is structured utilizing the [DeFiScan Framework](https://www.defiscan.info/framework), evaluating the protocol across five core dimensions: Chain, Upgradeability, Autonomy, Exit Window, and Accessibility.
